Java学习:使用DOM写XML -- 绘制中国国旗的SVG文件

利用 Flag.java文件绘制出一个SVG格式的中华人民共和国国旗。SVG文件参考了维基百科上的中国国旗的SVG文件

源码已push至GitHub FlagPRC

Java Code
